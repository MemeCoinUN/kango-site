## Deploy (GitHub Pages)
1. Create a repo (e.g. `kango-site`).
2. Upload all files from this folder.
3. In Settings → Pages → set **Branch: main** and **Folder: /(root)**.
4. Your site will be live at:
   https://YOUR_GITHUB_USERNAME.github.io/kango-site/
